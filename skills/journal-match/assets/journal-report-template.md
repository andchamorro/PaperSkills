# Journal Report Template — HTML Structure

This template defines the journal-specific HTML structure for the recommendation report. Apply all CSS from `skills/shared/report-template.md` as the base design system. The components below extend that system for journal-match reports.

---

## HTML Skeleton

```html
<!DOCTYPE html>
<html lang="{{LANG}}">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Journal Recommendations — {{MANUSCRIPT_TITLE}}</title>
  <!-- Font loading per report-template.md (English or Chinese variants) -->
  <style>
    /* Base CSS from report-template.md */
    /* Journal-specific extensions below */
  </style>
</head>
<body>
  <div class="report">

    <!-- HEADER -->
    <header class="report-header">
      <p class="report-tag">Journal Recommendations</p>
      <h1 class="report-title">{{MANUSCRIPT_TITLE}}</h1>
      <p class="report-subtitle">Target venue analysis based on scope, methodology, and impact</p>
      <div class="report-meta">
        <span>PaperSkills</span>
        <span class="meta-sep">·</span>
        <span>{{DATE}}</span>
      </div>
    </header>

    <!-- STATS BAR -->
    <div class="metrics">
      <div class="metric-card">
        <div class="metric-value">{{JOURNAL_COUNT}}</div>
        <div class="metric-label">Journals Analyzed</div>
      </div>
      <div class="metric-card">
        <div class="metric-value">{{BEST_TIER}}</div>
        <div class="metric-label">Best Match Tier</div>
      </div>
      <div class="metric-card">
        <div class="metric-value">{{OA_PERCENTAGE}}%</div>
        <div class="metric-label">Open Access</div>
      </div>
    </div>

    <!-- MANUSCRIPT PROFILE CALLOUT -->
    <div class="callout">
      <strong>Manuscript Profile</strong> —
      {{DISCIPLINE}} · {{SUB_FIELD}} · {{METHODOLOGY}} · {{SCOPE}}
      <br>Keywords: {{KEYWORDS}}
    </div>

    <!-- TIER 1: BEST MATCH -->
    <h2>Tier 1 — Best Match</h2>
    <!-- Repeat .journal-card for each Tier 1 journal -->
    <div class="journal-card">
      <div class="journal-card-header">
        <h3 class="journal-name">
          <a href="{{HOMEPAGE_URL}}">{{JOURNAL_NAME}}</a>
        </h3>
        <div class="journal-badges">
          <span class="tag">BEST MATCH</span>
          <!-- if OA: --><span class="tag tag-oa">OPEN ACCESS</span>
        </div>
      </div>
      <div class="journal-meta-row">
        <span>Scope: <strong>{{SCORE}}/5</strong> — {{SCORE_REASONING}}</span>
      </div>
      <div class="journal-meta-row">
        <span>H-index: {{H_INDEX}}</span>
        <span class="meta-sep">·</span>
        <span>Works: {{WORKS_COUNT}}</span>
        <span class="meta-sep">·</span>
        <span>Citations: {{CITED_BY_COUNT}}</span>
      </div>
      <div class="journal-meta-row">
        <span>Similar papers: {{SIMILAR_COUNT}} in last 3 years</span>
      </div>
      <p class="journal-notable">
        Notable: "{{RELATED_PAPER_TITLE}}" closely related to this manuscript
      </p>
    </div>

    <!-- TIER 2: GOOD ALTERNATIVE -->
    <h2>Tier 2 — Good Alternative</h2>
    <!-- Repeat .journal-card for each Tier 2 journal -->

    <!-- TIER 3: SPECIALIZED/NICHE -->
    <h2>Tier 3 — Specialized / Niche</h2>
    <!-- Repeat .journal-card for each Tier 3 journal -->

    <!-- LANGUAGE-SPECIFIC OPTIONS (only if non-English manuscript) -->
    <h2>Language-Specific Options</h2>
    <p>Journals publishing in {{LANGUAGE}} with regional academic impact:</p>
    <!-- Repeat .journal-card or simpler list for language-specific journals -->

    <!-- FOOTER -->
    <footer class="report-footer">
      <p>
        Recommendations based on venue analysis of similar papers via
        Semantic Scholar and OpenAlex. Scope scores reflect editorial
        alignment, not acceptance probability.
      </p>
      <p>Generated by PaperSkills · {{DATE}}</p>
    </footer>

  </div>
</body>
</html>
```

## Journal-Specific CSS Extensions

Add these styles alongside the base report-template.md CSS:

```css
/* Journal card */
.journal-card {
  background: var(--surface);
  border: 1px solid var(--border);
  border-radius: 4px;
  padding: 1.5rem;
  margin: 1rem 0;
}

.journal-card-header {
  display: flex;
  justify-content: space-between;
  align-items: flex-start;
  flex-wrap: wrap;
  gap: 0.5rem;
  margin-bottom: 0.75rem;
}

.journal-name {
  font-family: 'Source Sans 3', sans-serif;
  font-size: 1.1rem;
  font-weight: 600;
  color: var(--ink);
  margin: 0;
}

.journal-name a {
  color: var(--ink);
  text-decoration: none;
  border-bottom: 1px solid var(--border);
}

.journal-name a:hover {
  border-bottom-color: var(--accent);
}

.journal-badges {
  display: flex;
  gap: 0.35rem;
  flex-wrap: wrap;
}

.tag-oa {
  background: #E8F5E9;
  color: var(--positive);
}

.journal-meta-row {
  font-size: 0.9rem;
  color: var(--text);
  margin-bottom: 0.35rem;
}

.journal-notable {
  font-size: 0.9rem;
  color: var(--muted);
  font-style: italic;
  margin: 0.5rem 0 0;
}
```

### Chinese Variant Overrides

When generating in Chinese, apply these text substitutions:

| English | Chinese |
|---|---|
| Journal Recommendations | 期刊推荐 |
| Journals Analyzed | 分析期刊数 |
| Best Match Tier | 最佳匹配层级 |
| Open Access | 开放获取比例 |
| Tier 1 — Best Match | 第一层级 — 最佳匹配 |
| Tier 2 — Good Alternative | 第二层级 — 优良替代 |
| Tier 3 — Specialized / Niche | 第三层级 — 专业/细分 |
| Language-Specific Options | 语言特定选项 |
| BEST MATCH | 最佳匹配 |
| OPEN ACCESS | 开放获取 |
| Scope | 范围匹配 |
| H-index | H指数 |
| Works | 论文数 |
| Citations | 被引次数 |
| Similar papers | 相似论文 |
