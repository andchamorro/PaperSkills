# CVPR 2025 Conference Guidelines

## Page Limit

- Main paper: 8 pages (excluding references)
- References: Unlimited additional pages
- Supplementary material: Optional, separate PDF

## Formatting Requirements

- Two-column format
- 10pt Times font
- Letter paper size (8.5" x 11")
- 1" margins on all sides
- Figures and tables must fit within column width or span both columns

## Mandatory Sections

1. Abstract (max 300 words)
2. Introduction
3. Related Work
4. Method/Approach
5. Experiments
6. Conclusion

## Submission Requirements

- Anonymous submission (double-blind review)
- No author names or affiliations in main paper
- No acknowledgements in submission version
- Self-citations in third person
- PDF format only
- Maximum file size: 100MB

## Citation Format

- Use the IEEE citation style
- Citations should be numbered in square brackets
- Example: [1], [2, 3], [4-7]

## Figure Guidelines

- Minimum 300 DPI resolution
- Clear, readable labels
- Captions below figures
- Reference all figures in text
